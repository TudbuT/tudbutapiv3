TudbuTAPIv3 Documentation

TudbuTAPIv3 is an API for Minecraft client modifications, used to track usage, authenticate users, and deliver direct or client-wide messages.

Admin methods

Jump to content The following can only be used with the password set by the owner of the API:
  • POST /service/[service_name: str]/create { pass: str, servicePass: str }
  • /!\ POST /service/[service_name: str]/rename { pass: str, new: str }
  • /!\ POST /service/[service_name: str]/delete { pass: str }
/!\ The delete and rename methods will stop the API for a moment as they require a data migration during which the API would be unstable if it was kept running. This is why they are owner-only.


The following can be used by any service admin, which is someone who knows a servicePass.
  • POST /service/[service_name: str]/allowChat { servicePass: str, allow: bool }
  • POST /service/[service_name: str]/setPremium { servicePass: str, uuid: UUID, status: int }
  • POST /service/[service_name: str]/data { servicePass: str, uuid: Optional:UUID, data: JSON }
  • POST /service/[service_name: str]/data/send { servicePass: str, uuid: UUID, data: JSON }
  • POST /service/[service_name: str]/data/sendAll { servicePass: str, data: JSON }
